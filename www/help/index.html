\Toolbar\

  • UNDO - Undoes the last action on each press
  • REDO - Redoes the last undone action on each press
  • PAN - Enables workspace panning. Circuit interaction is disabled
  • EDIT - Enables circuit editing. Panning is disabled
  • SAVE - Saves the current circuit into a loadable .json file
  • LOAD - Loads the provided .json file into the workspace
  • DELETE - Deletes the entire circuit from the workspace after confirmation
  • HELP - Redirects to the help page for help with the simulator
