SNEEZYhelp

\Side Panel\

  • COLLAPSING - Clicking any of the component headers will collapse that menu
  • ADDING TO WORKSPACE - While in EDIT mode, drag and drop components from the side panel to the workspace to begin using them
  • ACTIVATING - While in EDIT mode, clicking any component will activate it (if it is an input)

\Toolbar\

  • PAN - Enables workspace panning. Circuit interaction is disabled
  • EDIT - Enables circuit editing. Panning is disabled
  • DELETE - Deletes the entire circuit from the workspace after confirmation
  • HELP - Redirects to the help page for help with the simulator

\Mouse\

  • PANNING - While in PAN mode, click and drag to pan the circuit
  • ZOOMING - While in PAN mode, use the scroll wheel to zoom in and out
  • ACTIVATING INPUTS - While in EDIT mode, clicking any input component will activate it
  • WIRES - While in EDIT mode, clicking any two connection points will create a wire connection between them. Invalid connections will not work.
  • SELECTING - While in EDIT mode, double-clicking any component will select it

\Keyboard\

  • DELETE - Pressing while any component is selected will delete it
  • BACKSPACE - Pressing while any component is selected will delete it
